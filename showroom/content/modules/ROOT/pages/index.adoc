= How to Use This Guide

Welcome to the *CertChain Blockchain Credentialing* demo on Red Hat OpenShift.

This guide walks you through the demo environment — from issuing academic certificates on a Hyperledger Fabric blockchain to verifying them through a public portal.

== What You'll Need

* Access to the OpenShift Console (see the *OpenShift Console* tab above)
* Access to the embedded Terminal (see the *Terminal* tab above)
* A modern web browser

== Demo Flow

. *Admin: Issue Certificates* — Log in to a Course Manager UI, create a course, enroll students, and issue blockchain-backed certificates.
. *User: Verify Certificates* — Use the public Cert Portal to verify certificates by ID.
. *Explore Monitoring* — View Fabric network and application metrics in Grafana.
. *API Walkthrough* — Use `curl` to interact directly with the REST APIs.
